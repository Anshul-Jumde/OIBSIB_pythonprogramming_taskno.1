# OIBSIB_pythonprogramming_taskno.1

## BMI Health Tracker

## Description

Smart BMI Health Tracker is a Python GUI application developed using CustomTkinter. It calculates Body Mass Index (BMI) using the user's weight and height and classifies the result into different health categories.

## Features

* GUI using CustomTkinter
* BMI Calculation
* Health Category Detection
* Input Validation
* Health Suggestions
* Clear Button
* Dynamic Background Colors
* User-Friendly Interface

## Technologies Used

* Python
* Tkinter

## Outcome

Successfully developed a BMI Calculator application that calculates the Body Mass Index (BMI) based on user-provided height and weight. The application accurately determines the BMI category and provides users with a simple and interactive interface for health assessment.
